# AOTY Crawler UI

Streamlit-based dashboard for exploring AOTY scraped data.

## Quick Start

```bash
cd aoty-crawler/ui
pip install -r requirements.txt
streamlit run app.py
```

## Features

- **Search Dashboard**: Filter by genres, scores, year
- **Statistics Overview**: Total albums, unique genres, top albums
- **Album Grid**: Visual album cards with covers
- **Genre Distribution**: Text-based genre breakdown
